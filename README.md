# DTimestamp Overlay

DTimestamp Overlay is a Chrome extension (Manifest V3) that injects a live date, time, and timezone overlay onto every browser tab. It is designed specifically for screenshot timestamping and is security-hardened to meet SOC 1 and SOC 2 standards.

## Features

* **Live Overlay:** Displays real-time date, time, and timezone directly on your active browser tabs.
* **Modern Architecture:** Built using Chrome's latest Manifest V3 for optimal performance and security.
* **Compliance Ready:** Security-hardened to SOC 1 and SOC 2 standards, making it ideal for audit and compliance screenshots.
* **Lightweight:** Primarily built with JavaScript and HTML for minimal browser impact.

## Installation

To install the extension locally in Google Chrome:

1. Download the `dtimestamp_overlay.zip` file from this repository or clone the repository.
2. Extract the contents of the zip file.
3. Open Google Chrome and navigate to `chrome://extensions/`.
4. Enable **Developer mode** using the toggle in the top right corner.
5. Click the **Load unpacked** button and select the extracted `dtimestamp_overlay` folder.

## Usage

Once installed and enabled, the extension automatically injects the timestamp overlay onto your web pages. Simply navigate to the desired page and take your screenshot to capture the verified time and date.
